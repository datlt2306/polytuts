# Cài đặt Reactjs

React đã được thiết kế để dễ dàng tích hợp vào các dự án từ nhỏ đến lớn. Bạn có thể sử dụng React ít hay nhiều tùy theo nhu cầu của mình. Dù bạn chỉ muốn khám phá React, thêm một chút tính tương tác vào trang HTML hay bắt đầu một ứng dụng phức tạp được sử dụng bởi React, trong phần này tôi sẽ giúp bạn bắt đầu học một cách dễ dàng, từng bước một. Ok bắt đầu nào

:::note Nội dung

-   Cách để tạo một ứng dụng React mới từ HTML page
-   Cách bắt đầu một dự án React độc lập
